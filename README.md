This repository contains Maple code implementing the algorithms described in:
G. M. Díaz et al., Relative position of a parabola or a hyperbola and an ellipse without computing intersection points,(https://doi.org/10.48550/arXiv.2505.00706)
Requirements

	•	Maple 6 or later
Files

	•	parabcircPublic.txt: Maple code for determining the relative position between an ellipse and a parabola.
	•	hypercircPublic.txt: Maple code for determining the relative position between an ellipse and a hyperbola.
	•	parab_ellipsePublic.mw: examples illustrating the use of parabcircPublic.txt
	•	parab_hipPublic.mw: examples illustrating the use of hypercircPublic.txt
